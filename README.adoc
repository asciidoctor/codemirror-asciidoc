# AsciiDoc mode for CodeMirror

This repository contains the AsciiDoc mode for CodeMirror.

## Installation

```
$ npm install codemirror-asciidoc
```

## Usage

```js
var cm = ...;
cm.defineMode('asciidoc', require('codemirror-asciidoc'));
```
